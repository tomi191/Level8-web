# LEVEL 8 — Digital Agency Website

Premium dark-mode agency website for **LEVEL 8 EOOD** — a Bulgarian digital agency specializing in e-commerce, AI automation, and loyalty programs.

## Tech Stack

- **Next.js 16** (App Router, Turbopack)
- **Tailwind CSS v4** (CSS-first configuration)
- **shadcn/ui** components
- **motion/react** (Framer Motion) animations
- **TypeScript** strict mode

## Features

- Neon-green terminal/hacker aesthetic with scanline overlays
- Interactive AI chatbot widget with guided conversation flow
- Portfolio marquee with browser-mockup cards (pause on hover)
- Pricing cards with horizontal scroll-snap on mobile
- Tech stack infinite marquee
- SpotlightCard hover effects with cursor-tracking gradient
- DecryptedText animation on headings
- Scroll-spy navigation with active section highlighting
- Lead magnet terminal-style email capture form
- Contact form with Zod v4 validation
- Full mobile optimization (390px+ viewports)
- SEO: sitemap, manifest, Open Graph, structured data
- Accessibility: skip-nav, aria-labels, reduced-motion support

## Development

```bash
pnpm install
pnpm dev    # runs on port 3003
```

## Deployment

Deployed on [Vercel](https://vercel.com) — auto-deploys on push to `master`.
